# Ally & Dalton Wedding Website Starter

Static starter website for Ally Hall and Dalton Vaughan's wedding website.

## Files

- `index.html` — website structure/content
- `styles.css` — colors, typography, layout, mobile responsiveness, animations
- `script.js` — mobile menu, header scroll state, reveal animations
- `assets/images/` — place photos and graphics here

## GitHub Pages setup

1. Create a new GitHub repository.
2. Upload all files/folders in this package to the **root** of the repository.
3. Go to **Settings → Pages**.
4. Set:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Wait a minute or two for GitHub Pages to publish the site.

## Quick edits

- Colors are in `styles.css` under `:root`.
- Main page content is in `index.html`.
- Replace placeholder gallery blocks with real images inside the Gallery section.
- Replace `href="#"` for RSVP and Registry buttons with your real links later.
